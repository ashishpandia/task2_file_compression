# Task 2 - File Compression Tool

## Objective
Build a C++ application to compress and decompress files.

## Description
This project demonstrates a simple file compression technique using Run Length Encoding (RLE). The program reads data from a text file, compresses repeated characters, and stores the compressed output in another file.

## Features
- Read data from file
- Compress text data
- Save compressed output
- Easy to understand implementation
- File handling in C++

## Technologies Used
- C++
- File Handling (fstream)

## Input Example
AAAABBBBCCCCDD

## Compressed Output
A4B4C4D2
